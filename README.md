# htpq.github.io — istruzioni rapide

Questo repository pubblica una semplice pagina statica su GitHub Pages.

Come modificare
- Apri il file (es. index.html) nella web UI di GitHub → clicca l'icona a matita per modificare → Commit changes.
- Le modifiche si riflettono sul sito dopo pochi minuti.

File inclusi
- index.html — pagina principale
- styles.css — stili
- README.md — istruzioni
- .gitignore — pattern comuni da ignorare

Sicurezza raccomandata
- Abilita l'autenticazione a due fattori (2FA) nel tuo account GitHub.
- Non caricare chiavi private o file sensibili.
- Mantieni il repository pubblico per usare Pages su username.github.io.
